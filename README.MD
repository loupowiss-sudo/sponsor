# Hichem Sponsor Dashboard

Application de gestion de sponsoring et espace client.

## Fonctionnalités

- **Espace Administrateur** : Gestion des clients, transactions, offres et statistiques.
- **Espace Employé** : Accès restreint pour la gestion quotidienne (sans accès aux profits).
- **Espace Client** : Suivi des demandes, méthodes de travail et paiements.

## Technologies

- HTML5 / JavaScript (Vanilla)
- Tailwind CSS (via CDN)
- Firebase (Auth, Firestore)

## Installation

Il suffit d'ouvrir le fichier `index.html` dans un navigateur web moderne.
Aucune installation de serveur n'est nécessaire (architecture Serverless).
